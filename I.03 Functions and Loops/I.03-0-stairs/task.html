I.03-0 Лесенка

Задание: Пользователь вводит с клавиатуры некоторый текст. Программа в ответ начинает постепенно его уничтожать: по очереди удаляет символы (начинает с первого) и выводит результат в консоль (через пробел).

Пример 1:
Вводим:
ЪУЪ
Получаем:
ЪУЪ ЪУ Ъ

Пример 2:
Вводим:
Дратути
Получаем:
Дратути Дратут Драту Друт Дра Др Д

См. методы по работе со строками здесь.
