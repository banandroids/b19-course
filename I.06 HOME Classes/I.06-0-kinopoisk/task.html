I.06-0 Задание от Кинопоиска I

Задание: Кинопоиск пишет базу данных на Котлине для хранения фильмов и просит вас создать класс фильма Film с такими свойствами (именно в таком порядке):

  • name (строка) - название фильма, по умолчанию null
  • year (целое число Int) - год выхода фильма, по умолчанию 0
  • rating (дробное число Float) - рейтинг фильма, по умолчанию 0
  • reviews (массив строк) - рецензии на фильм, по умолчанию пустой массив
  • favourite (true или false) - добавил ли пользователь фильм в избранное, по умолчанию false
