I.06-2 Срочное задание от Last.fm

Задание: Музыкальный сервис Last.fm просит дописать в main код чтения песен из консоли: в консоль пользователь вводит сначала количество песен, а затем по очереди для каждой песни - название песни, имя исполнителя и год выпуска.

Все эти прочитанные песни нужно поместить в массив songs (при этом предварительно его нужно проинициализировать с нужным размером).

Пример 1:
Вводим:
2
bad guy
Billie Eilish
2019
Yesterday
The Beatles
1995

Получаем: массив songs должен состоять из двух элементов: первый - песня "bad guy", второй - "Yesterday"

Пример 2:
Вводим:
0

Получаем: массив songs должен быть пустым (то есть ноль элементов)

Сначала нужно запросить размер, потом записать в songs пустой массив нужного размера, а потом сколько песен - сколько раз считать с консоли нужные свойства и закинуть в массив песню.
