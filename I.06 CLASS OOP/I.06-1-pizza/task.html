I.06-1 Пиццерия

Задание: Одна пока малоизвестная пиццерия тоже переходит на Котлин и поэтому просит вас на основе готового класса Pizza написать три метода (то есть каждый метод должен возвратить объект класса Pizza с нужными свойствами):

  • метод makeMargarita - для приготовления пиццы "Маргарита", в состав которой входит томатный соус, сыр "моцарелла" и базилик
  • метод makePepperoni - для приготовления пиццы "Пепперони", в состав которой входит томатный соус, сыр "моцарелла" и ровно 7 ломтиков пепперони
  • метод makeQuattroFormaggio - для приготовления пиццы "Четыре сыра", в состав которой входит только четыре сыра "моцарелла" "пармезан", "дорблю", "эмменталь"
