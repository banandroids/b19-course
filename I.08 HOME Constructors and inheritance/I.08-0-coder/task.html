I.08-0 Кодинг

Задание: Научите уже готовый класс Coder писать код и спать. У Coder пока есть два свойства: навык программирования skill (изначально 1) и усталость fatigue (изначально 0.0).

Нужно добавить два метода:

  • метод sleep с одним Int параметром hours: этот метод уменьшает усталость кодера на значение 0.1 * hours - при этом усталость не может быть отрицательной
  • метод writeCode:
    • если кодер устал (усталость равна 1), то метод возвращает строку "¯\_(ツ)_/¯"
    • если не устал, то кодер пишет код - строку, состоящую из N слов, разделённых пробелом, где N - навык программирования кодера, а каждое слово - рандомное из набора "val", "class", "fun" (например, со скиллом 4 кодер может вернуть "fun class val class")
    • метод увеличивает усталость на 0.2, при этом усталость не может быть больше 1 (то есь максимальная усталость - 1)
    • и увеличивает навык программирования на 1

Пример:
Если вызвать такой код:
val coder = Coder() // создаём кодера
println(coder.writeCode()) // кодер пишет код, отображаем этот код в консоли
coder.sleep(4) // даём кодеру поспать 4 часа
println(coder.fatigue) // выводим его усталость в консоль
println(coder.writeCode()) // ещё разочек пишет код и в консоль
println(coder.skill) // отображаем его уровень
 То в консоли может быть такое (может отличаться словами в коде, там же рандом): class
0
fun class
3
